== Overview
=== Для кого продукт

Rust разработчик / трейдер, алготрейдер.
- Умение писать и читать код на Rust, уровень Junior необходим.
- Опыт в трейдинге от 2х лет.

=== Что умеет

Для трейдера: анализ исторических данных, поиск условий на графике,
    системные уведомления о событиях (аномалиях), терминал с кастомными
    визуализациями (web-gui), симулятор рынка.

Для алготрейдера: библиотека трейдерских абстракций, исторические данные,
    событийно ориентированные стратегии, бэктестер, live трейдер,
    мониторинг состояния роботов (web-gui).

=== Описание модулей

==== Общее
avin_data
    Скачивание биржевых данных: свечи, тики, стакан, SuperCandles от
    moexalgo.
avin_core
    Слой базовых трейдерских абстракций: бар, таймфрейм, график,
    ордер, аккаунт...
avin_connect
    Подключение к Т-банку, запрос исторических данных, стрим свечей,
    тиков и стаканов, выставление ордеров.
avin_analyse
    Расчет функции распределения случайной величины. Относительная
    классификация значений случайной величины, приближенная к человеческому
    восприятию (маленький, средний, большой, очень большой, аномалия,
    черный лебедь, и тп, всего 15 градаций + упрощенная классификация на
    5 градаций).
avin_search
    Поиск ситуаций по условиям, делает отметки на графике, которые
    можно посмотреть в гуи. Без логики выставления ордеров и тп.
    Инструмент предварительной проверки гипотез перед созданием стратегий
    будь то для ручной или алго торговли.

==== Для трейдинга
avin_adviser
    Системные уведомление по заданным условиям (например 5 бычьих баров подряд
    или аномальный размер тика).
avin_simulator
    Простой симулятор рынка для оттачивания навыков торговли.
avin_terminal
    Кастомный терминал.

==== Для алготрейдинга
avin_strategy
    Шаблон (trait) для создания стратегий.
avin_tester
    Простой бэк-тестер.
avin_trader
    Модуль запуска роботов в live режиме.
